# RAG Play Competitive Notes

Evidence captured locally in `research/ragplay-screenshots/`.

## What RAG Play Does Well

- Clear four-stage RAG explanation.
- Interactive chunking controls.
- Browser-side embeddings and UMAP visualization.
- Semantic search scores.
- Prompt preview and Groq-backed generation.

## Gap For rag-lens

RAG Play is mostly a visual/single-session explainer. rag-lens should be a practical workbench:

- User docs and curated examples.
- Supabase-backed persistence.
- Full trace records.
- Session cleanup.
- Debug-oriented inspector.
- Experiment comparison.
