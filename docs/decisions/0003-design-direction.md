# Decision 0003: Supabase + Linear + Mintlify Visual Direction

## Decision

Use a custom design system inspired by Supabase, Linear, and Mintlify.

## Rationale

- Supabase matches the database/vector/tooling story.
- Linear matches the precise dark trace-inspector feel.
- Mintlify matches educational docs and developer readability.

## Consequence

The app should feel like a RAG observability workbench. It should not look like a RAG Play clone, a marketing landing page, or a dark-only terminal product.
